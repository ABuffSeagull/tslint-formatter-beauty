## Installation

Using Yarn:

```
yarn add --dev tslint-formatter-beauty
```

Using npm:

```
npm install --save-dev tslint-formatter-beauty
```

## Usage

### tslint CLI:

```
tslint --formatters-dir ./node_modules/tslint-formatter-beauty -t beauty -p .
```
